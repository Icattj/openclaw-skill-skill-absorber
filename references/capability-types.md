# Capability Types — Classification Taxonomy

When analyzing a repo, classify each feature into one of these types:

## Data Access
- **Web Scraping** — extract content from websites (Jina Reader, Playwright, Puppeteer)
- **API Integration** — call external APIs (Twitter, Reddit, YouTube, GitHub)
- **Database** — query/write databases (SQLite, Postgres, MongoDB)
- **File Processing** — parse/transform files (PDF, DOCX, images, video)
- **Search** — web search, semantic search, FTS (Tavily, Brave, Exa, SearXNG)

## Communication
- **Messaging** — send/receive on platforms (Telegram, Slack, Discord, WhatsApp)
- **Email** — send/read email (SMTP, IMAP, Gmail API)
- **Voice** — TTS, STT, voice calls (ElevenLabs, Whisper, Twilio)
- **Notifications** — push alerts (webhooks, mobile push)

## Intelligence
- **Memory** — persistent context across sessions (vector DB, file-based, FTS)
- **Learning Loop** — self-improving from experience (skill creation, pattern recognition)
- **User Modeling** — building understanding of user preferences over time
- **Planning** — multi-step task decomposition and execution
- **Research** — deep web research with synthesis (multi-source, iterative)

## Infrastructure
- **Deployment** — deploy apps/services (Docker, cloud, serverless)
- **Monitoring** — system health, alerts, dashboards
- **Scheduling** — cron, task queues, recurring jobs
- **Terminal/Shell** — execute commands, manage processes
- **Version Control** — git operations, PR management

## Content
- **Writing** — generate/edit text content (blog, docs, copy)
- **Design** — generate visuals, UI mockups, diagrams
- **Code Generation** — write/refactor code
- **Translation** — language translation
- **Summarization** — condense content (articles, videos, podcasts)

## Automation
- **Browser Automation** — interact with web UIs (Playwright, Selenium)
- **Workflow Orchestration** — chain multiple tools/agents
- **Data Pipeline** — ETL, data transformation chains
- **Testing** — automated testing, validation

## Analysis
- **OCR** — extract text from images/documents
- **Vision** — image/video understanding
- **Data Analysis** — statistical analysis, charting
- **Sentiment** — opinion/emotion detection
- **Competitive Intel** — market/competitor analysis
